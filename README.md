# README
このリポジトリは、麻雀ゲームの実装を目的としています。
node.jsを使用して、ブラウザ上で動作する麻雀ゲームを開発しています。
また、server側では、socket.ioを使用してリアルタイムな通信を実現しています。


## 麻雀ゲーム
麻雀ゲームの実装です。現在は三人麻雀、四人麻雀に対応しています。


## 実装済み機能
- 三人麻雀ではチー禁止（ポンのみ）
- 三人麻雀の北抜き（手牌→抜きドラ領域、ツモ補充）
- 北抜き枚数を抜きドラ（1枚＝1飜）として点数加算
- 役判定と飜数に応じた点数計算（基本符30固定の簡易版）
  - 立直・門前清自摸和・断么九・役牌・平和・対々和・三色同順・一気通貫・混老頭・七対子・抜きドラ
- 初期点数を30000点に統一
- リーチ宣言（1000点供託、テンパイ・門前・残り山4枚以上が必要）
- リーチ後の打牌制限（リーチ後は自由打牌不可）
- リーチ棒の供託・回収（次の和了者がまとめて獲得）
- 本場ボーナス（300点／本場 ロン時、100点／本場×3 ツモ時）
- 九種九牌の流局宣言（配牌直後、自分のターンで么九牌9種以上）
- Private Room（パスワード付きルームの作成・参加）
- 牌山シャッフルを `crypto.randomInt`（CSPRNG）で実施
- 全ファイルにTypeScript初学者向け解説コメント


## 未実装・既知の制限
- 符計算は省略（30符固定相当）
- 一発・裏ドラ・ダブルリーチ・国士無双・大三元等の役は未実装
- カン（暗槓・明槓）未実装
- 連荘ボーナスの詳細ルール未実装
- ゲーム中の切断ハンドリング簡略


## サーバーの環境変数
本番デプロイ時は必ず `ALLOWED_ORIGINS` を指定してください。未指定の場合は
ローカル開発用オリジン（localhost:5173/4173）のみ許可されます。

| 変数名 | 用途 | 例 |
|--------|------|----|
| `PORT` | リッスンポート | `3001` |
| `ALLOWED_ORIGINS` | CORS 許可オリジン（カンマ区切り） | `https://your-github-username.github.io,https://your-domain.example` |

設定例は [`server/.env.example`](server/.env.example) を参照。
