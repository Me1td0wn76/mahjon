# README
このリポジトリは、麻雀ゲームの実装を目的としています。  
node.jsを使用して、ブラウザ上で動作する麻雀ゲームを開発しています。  
また、server側では、socket.ioを使用してリアルタイムな通信を実現しています。  


## 麻雀ゲーム
麻雀ゲームの実装です。現在は三人麻雀、四人麻雀に対応しています。  


##　既存の不具合
- 三人麻雀の時にチーができる(ポンのみにする)
- 三人麻雀の時に北抜きできるようにする。
- 三人麻雀の時に北をドラ牌にする。
- 点数計算のチェックをする。
- 初期点数の設定を変える。
- リーチの宣言をしたときに点棒を出すようにする。
- Private Roomの実装をする。
- 九種九牌のルールを実装する。
- 牌の山をシャッフルする。
